🤖 AI Chatbot App (Android + Spring Boot + Ollama)

🚀 Overview
A full-stack AI chatbot application with a modern UI and local LLM integration.

🧱 Tech Stack
- Android (Kotlin)
- Spring Boot (Java)
- Ollama (Local LLM)
- Retrofit + OkHttp

✨ Features
- Real-time chat UI
- Typing effect
- Dark mode + glass UI
- Local AI (no API cost)

🧠 Architecture
Android → Spring Boot → Ollama → LLM → Response

💡 Learnings
- Handling LLM latency
- API integration
- Full-stack system design
